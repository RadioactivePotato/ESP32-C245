# ESP32-C245无线烙铁

#### Description
ESP32-C245无线烙铁是一款便携自带锂电池供电的无线式电烙铁  使用C245发热芯  支持TYPE-C 快充直接供电和给锂电池充电  支持烙铁温度  电流 电压 功率 等参数的实时测量和显示
